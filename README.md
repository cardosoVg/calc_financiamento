# 📊 Calculadora de Financiamento

Ferramenta simple de simulação de financiamentos, ideal para comparar taxas e entender o custo total de juros ao longo do tempo.

## ✨ Funcionalidades

- Calculo de valor das parcelas mensais.
- Montante total pago ao final do contrato.
- Comparativo de juros totais.
- Ponto de virada entre os preços de SAC e Price

## 🚀 Como usar (Windows)

1. Vá na aba [Releases](https://github.com/cardosoVg/calc-financiamento/releases).
2. Baixe o arquivo `CalcFinanciamento.exe`.

## 🛠️ Para Desenvolvedores (

```zsh
# Clonar o repositório
git clone https://github.com/cardosoVg/calc_financiamento.git
cd calc_financiamento

# Instalar dependências 
pip install -r requirements.txt

# Rodar
python financiamento.py
```
